# CORE WORLD RULES

- planet’s **core is main light + heat source**
- surface world = **floating islands orbiting above it**
- habitable zone sits **above cloud layer**
- cloud layer = **“cloud sea” equivalent of ocean level**
- light mostly comes from
    - scattered core light
    - atmospheric piping
    - reflections
    - maybe bright moon(s)

result = **permanent twilight world**

---

# ATMOSPHERE + LIGHT

- brightest altitude = **just above clouds**
- higher than that → thinner air → dimmer harsher light
- below clouds → darker + hotter
- agriculture concentrates in **lightbelt band**

---

# WATER CYCLE

water comes upward via:

- fog rise
- storm updraft spray
- condensation at cloud tops

survival depends on terrain storing moisture:

- moss / porous rock = good
- bare rock + wind = dry

---

# ISLAND TYPES

### anchored islands

- barely move (0–5 km/month)
- oldest civilizations
- stable climates

### clusters

- archipelagos drifting together
- 20–300 km/month
- act like floating continents

### belt islands

- ride fast atmospheric lanes
- 200–2000 km/month along lane
- main trade network regions

### drifters

- small wandering fragments
- 500–5000 km/month
- chaotic paths
- ecological mixing

---

# ISLAND SPACING

large islands  
→ hundreds km apart

small islands (1–2 km²)  
→ 5–50 km apart  
→ dense swarms possible

touching is rare, but **close passes happen often**

ecosystems survive because dispersal happens via:

- wind
- birds
- storms
- drifting debris islands

---

# CONTACT FREQUENCY

same-lane islands → meet every few years  
adjacent-lane islands → every 1–6 months  
turbulent zones → weekly–monthly passes

trade routes = **timing windows**, not fixed paths

---

# SKY WEATHER

above clouds:

- steady directional winds
- jetstream-like lanes

belt intersections + calm zones = major cities + trade hubs

---

# MOONS

- can act as bright night lights
- create tidal pull on islands
- cause wobble / stress / altitude shifts

---

# WORLD SHAPE SUMMARY

the sky behaves like an ocean:

- belts = currents
- clusters = archipelagos
- drifters = floating debris
- anchored islands = seamounts

maps must include **time**, not just location

people don’t ask:

> where is it?

they ask:

> where will it be?
