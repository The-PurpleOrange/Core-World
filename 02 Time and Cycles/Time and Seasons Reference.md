CORE PRINCIPLE
Time is measured from sky behavior, not astronomy. The planet has layered cycles instead of a single global year.

GLOBAL TIME STRUCTURE

Smallest to largest units:

Day
Lightcycle (day + night from shadow belts)
Season Phase
Turning
Great Cycle

DAY / NIGHT

Cause:
Rotating particulate shadow belts between core and islands

Effect:
They periodically block core light and create global night

Properties:

Same everywhere on planet

Reliable and predictable

Defines universal clock

This is the only time unit all cultures agree on.

THE TURNING

Length:
~240 days

Definition:
One full atmospheric seasonal cycle

Each Turning always passes through the same six phases:

Green
Bright
Stormrise
Still
Crosswinds
Passing

A Turning is the closest equivalent to a “year”.

THE SIX SEASONS

These are atmospheric states, not temperature seasons.

Green
Thick clouds, high moisture, strong growth, safest travel

Bright
Thinner clouds, higher light, best visibility, trade season

Stormrise
Vertical storms, violent weather, floods, dangerous travel

Still
Low wind, stagnant air, quiet sky, calm travel

Crosswinds
Strong lateral winds, turbulent sky, migration and dispersal

Passing
Island alignments, shifting tides, trade and conflict season

SEASON LENGTH

Phases vary in length but total ≈ 240 days.

Typical ranges:

Green 55–75 days
Bright 35–55 days
Stormrise 10–25 days
Still 35–55 days
Crosswinds 10–25 days
Passing 10–25 days

Stable regions have predictable seasons.
Drifting regions experience irregular phase lengths.

THE GREAT CYCLE

Length:
1920 days

Definition:
Time required for a belt region to return to same global position AND same seasonal phase.

Equivalent to:
8 Turnings

This is the planet’s true long-term cycle.

DRIFT + SEASONS RELATIONSHIP

The slower an island moves:
the more stable its seasons.

Anchored islands → predictable climate
Belt islands → regular shifting seasons
Clusters → semi-regular
Drifters → chaotic, short “seasons”

WHAT CONTROLS SEASONS

Not axial tilt.

Seasons are controlled by:
atmospheric circulation
cloud density
vertical convection
moon tidal influence
island drift position

WHAT MOONS DO

Moons do not create day/night.

They affect:
storm timing
tides in cloud sea
belt drift timing
season start shifts

So moons modify timing, not light source.

HOW DATES ARE WRITTEN

Standard format:

Day number + Phase + Turning + Great Cycle

Example:
Day 73, Bright Phase — 5th Turning, 3rd Great Cycle

CULTURAL TIMEKEEPING RULE

Global coordination uses:
day count

Local planning uses:
season phase

Meaning:
Different regions may call different days “new year” because they track seasons, not Turnings.

MASTER WORLD RULES
